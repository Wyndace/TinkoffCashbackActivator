# TinkoffCashbackActivator

Chrome-расширение, которое добавляет кнопку **«Активировать весь кешбэк»** на страницу Тинькофф Банка и активирует все неактивные кешбэк-категории одним кликом.

## Установка

1. Скачай [`TinkoffCashbackActivator.crx`](TinkoffCashbackActivator.crx)
2. Открой `chrome://extensions/`
3. Включи режим разработчика
4. Перетащи `.crx` в окно расширений

## Как работает

Расширение инжектит кнопку на страницу. При клике — находит все кнопки активации кешбэка и программно нажимает их.

## Стек

- Vanilla JS (content script)
- Chrome Extensions Manifest V2
